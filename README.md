Codigos de codewars
===============================================

En este repositorio están todos los códigos que he hecho por mi cuenta durante todo el curso.
---

1. **Codigo `strings_mix`** ["code"](./src/strings_mix.py)
   Compara dos strings y devuelve qué letras minúsculas aparecen más veces en cada uno.

2. **Código `solution`** ["code"](./src/mean_square_error.py)  
   Calcula la media de los cuadrados de las diferencias entre dos arrays de números. Devuelve 0 si los arrays están vacíos.

3. **Código `pig_it`** ["code"](./src/pig_latin.py)  
   Convierte un texto a “Pig Latin”: mueve la primera letra de cada palabra al final y añade "ay". Mantiene signos de puntuación sin cambios.

4. **Código `time_stamp`** ["code"](./src/time_stamp.py)  
   Convierte una fecha y hora (`year, month, day, hour, minute, second`) a segundos desde el 1 de enero de 1970 (Unix epoch), considerando años bisiestos.

5. **Código `find_outlier`** ["code"](./src/find_outlier.py)  
   Encuentra el número par o impar que es distinto en un array donde la mayoría de los números tienen la misma paridad.

6. **Código `first_n_smallest`** ["code"](./src/n_smallest.py)  
   Devuelve los `n` números más pequeños de un array manteniendo el **orden original** de aparición.

7. **Código `narcissistic`** ["code"](./src/narcissistic.py)  
   Comprueba si un número es narcisista (Armstrong), es decir, si la suma de sus dígitos elevados a la cantidad de dígitos es igual al propio número.
